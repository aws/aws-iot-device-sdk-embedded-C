Overview

The AWS IoT Device SDK for Embedded C provides demonstration code and integration tests for a collection of libraries.

Libraries

The following libraries (along with their source repositories) are part of this SDK:

  • MQTT client - coreMQTT
  • HTTP client - coreHTTP
  • JSON parser - coreJSON
  • PKCS #11 implementation - corePKCS11
  • AWS IoT Device Shadow client - device-shadow-for-aws-iot-embedded-sdk
  • AWS IoT Device Defender client - device-defender-for-aws-iot-embedded-sdk
  • AWS IoT Jobs client - jobs-for-aws-iot-embedded-sdk
  • AWS IoT Fleet Provisioning Library - fleet-provisioning-for-aws-iot-embedded-sdk
  • AWS SigV4 Library - sigv4-for-aws-iot-embedded-sdk
